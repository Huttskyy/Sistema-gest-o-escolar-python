# Sistema de Gestão Escolar em Python

## Sobre o Projeto

Este projeto consiste em um sistema de gestão escolar desenvolvido em Python. O programa permite o cadastro de notas e faltas de alunos em diferentes disciplinas, calculando as médias e determinando a situação acadêmica de cada estudante.

Além disso, o sistema gera relatórios detalhados e um resumo geral contendo a quantidade de alunos aprovados e reprovados.

## Funcionalidades

* Cadastro de notas de alunos.
* Validação de entradas numéricas.
* Controle de faltas por disciplina.
* Cálculo automático de médias.
* Verificação de aprovação ou reprovação.
* Relatório individual por aluno.
* Relatório geral da turma.

## Disciplinas Cadastradas

* Português
* Matemática
* Biologia
* Química
* Física

## Regras de Aprovação

### Aprovação por Nota

O aluno é aprovado na disciplina quando sua média é maior ou igual a 6,0.

### Reprovação por Nota

O aluno é reprovado quando sua média é inferior a 6,0.

### Reprovação por Faltas

O aluno é automaticamente reprovado caso possua 51 faltas ou mais em uma disciplina.

## Tecnologias Utilizadas

* Python 3

## Exemplo de Uso

1. Selecione a opção "Cadastrar notas".
2. Informe as notas e faltas dos alunos.
3. Gere o relatório final através do menu.
4. Visualize os resultados individuais e o resumo geral.
